Predict car fuel efficiency (MPG) using a Random Forest regression model.

Features
  Uses features: Cylinders, Horsepower, Displacement
  
  Interactive Streamlit app for MPG prediction
  
  Trained model saved as mpg.pkl

Files
  app.py — Streamlit app code
  
  Autompg_regression.ipynb — Data analysis and model training
  
  mpg.pkl — Saved model
  
  mpg_cleaned.csv — Cleaned dataset
  
  requirements.txt — Python dependencies

How to run
  Install dependencies:
    pip install -r requirements.txt
  Run the app:
    streamlit run app.py
